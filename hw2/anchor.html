The <a> tag

  • <a> stands for anchor.
  • <a> defines a hyperlink, which is most commonly used to link from one page to another.

Required Attributes:

href - specifies a URL which will be the target of the hyperlink

Other Attributes Example

You can create a place holder link by leaving href attributes empty:

<a href="">link</a>

You can create a direct link to other pages in the web:

<a href="http://www.w3.org/TR/html5/text-level-semantics.html#text-level-semantics">link</a>

You can create a link which opens content in a new tab or new window

<a href="anchor.html" target="_blank">link<a>

You can create a downloadable link

<a href="anchor.html" download>link</a>
