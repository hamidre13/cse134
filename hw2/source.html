The <source> tag

The <source> element indicates the source of multimedia content that is included in <audio>, <picture>, and <video> tags. You can specify multiple sources, and a browser will choose one that works correctly. If none works, the content inside the multimedia tag is displayed instead.

Example 1

<source> elements nested inside a <picture> element describe multiple sources the content in the <img> tag can come from. Typically the 'media' attribute is defined. It contains a boolean expression that, when evaluated to 'true', indicates that this source element is the one to use. Resize the window to see the source change!
