The <col> tag

  • <col> defines a column within a table
  • <col> can be use to define some common semantics (like span) of the cell in the column with the use of different attributes
  • <col> in HTML4 can be use to define semantics such as alignment, color, width...etc. However,HTML5, use css for those modification
  • <col> will most likely be found within <colgroup> tag
