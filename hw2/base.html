The <baes> tag

  • <base> defines a base URL of all relative URLs within a document
  • <base> element can only be define once in a html document
  • <base> element should be contain inside <head> element
  • <base> can be use to pre-define attributes that will apply to all hyperlink within the document, regardless of whether base URL are used
