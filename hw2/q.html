The <q> tag

The <q> tag is used to indicate a short quotation.

Example 1

Mark Twain once said, Go to heaven for the climate, hell for the company.

Example 2

This quotation includes the 'cite' attribute, which specifies the source of the quotation. It has no visual effect, but it can be utilized by some screen readers.

Mark Twain once said I can live for two months on a good compliment.
