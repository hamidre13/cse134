The <abbr> tag

  • <abbr> stands for abbreviation.
  • <abbr> is used to define a series of letter as the short-hand of a string of words.
  • <abbr> provides useful information to browsers, translation systems and search-engines. For example, in some browsers, when the user hover the cursor onto the abbreviation, the string of words defined in the title attributes will appear as a tooltip. However, not wrapping an abbreviation using <abbr> tag does not make a significant difference in terms of initial look.

Example:

WWW stands for World Wide Web

<abbr title="World Wide Web">stands for World Wide Web<abbr>
