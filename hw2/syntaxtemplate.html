<input type="text"> Input

Input tags will be used to collect information from user. This information can be processed either locally or on the server.If type does not get specified thedefalut type will be set which is text.

Specification

XML DTD Syntax

DTD only (without attr list)

					
						<ENTITY % InputType
						  "(TEXT | PASSWORD | CHECKBOX |
						    RADIO | SUBMIT | RESET |
						    FILE | HIDDEN | IMAGE | BUTTON|
						    datetime |datetime-local |date|
						    month|time|week|number|range |email|url |search|tel|color )"
						   >

Syntax

				
					<input
					type="TEXT | PASSWORD | CHECKBOX |
					    RADIO | SUBMIT | RESET |
					    FILE | HIDDEN | IMAGE | BUTTON|
					    datetime |datetime-local |date|
					    month|time|week|number|range |email|url |search|tel|color" 
					     GlobalAttribuites 
					    name = "string"
					    disabled = ""|"disabled" 
					    form= "id" 
					    maxlength = "non negative intiger" 
					    readonly = "readonly"|"" 
					    size = "positive integer" 
					    value = ""
					    autocomplete = "on"|'off' 
					    autofocus = "autofocus"|"" 
					    list ="id" 
					    pattern = "" 
					    required = "required" | "" 
					    placeholder = 
					    dirname = 
					/ >

Example

The following code : 

							<label for="name" > Name : </label>
							<input type="text" dir="rtl" required="required" placeholder="This is a place holder" value="توماس" autocomplete = "on" name= "name">
gives us :

Element Attributes

disabled
Represents disabled control
form
The form id that this elment is inside it
maxlength
the maximum lenth
readonly
the form elemnt is not editable
size
The number of charachters that will be shown to user agent in browsing area.
value
Default vlaue for the input,(Any string that contains no line feed)
autocomplete
Browser can complete the form or not
autofocus
The first elemtnt ot focus to as soon as the document gets loaded
list
THe list value id wich this input is associated with
pattern
Regex pattern that validates the form input
required
Specifies that the input should be filled in order for the form to gets submitted"
placeholder
Hint on how to fill out the input
dirname
Enables for right to left languages to have the attr name with the rtl value
