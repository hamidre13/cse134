The <bdo> tag

  • <bdo> stands for bidirectional override element
  • <bdo> is used to set and override directionality of text enclosed
  • <bdo> uses attributes dir to set direction into rtl(right to left) or ltr(left to right)

Example

The following text is not overrided by <bdo>

Hello World!

The following text is override by <bdo> into right-to-left direction

Hello World!
