The <tfoot> tag

The <tfoot> element defines a set of rows (usually at the foot of the table) that are supposed to summarize the columns of the table.

Example 1

The foot of this table is styled differently because a style was applied to the <tfoot> tag.

Name Age
John Smith Jane Doe
21 23
Info More Info

Example 2

In this example, the <tfoot> tag is placed above the body and below the footer, but it still renders at the bottom of the table.

Name Age
Info More Info
John Smith Jane Doe
21 23
