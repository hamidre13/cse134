The <applet> tag

  • <applet> identifies an embedded java applet
  • <applet> has been removed in HTML5. <object> or <embed> should be use instead
  • <applet> tag use is discouraged because it is not supported by many common browsers

Required Attributes (Either or)

  • code - Specifies the URL of applet file to be loaded (a java .class file)
  • object - Specifies the URL of a serialized representation of an object
