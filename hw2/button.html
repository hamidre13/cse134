The <button> tag

  • <button> tag defines a clickable button
  • <button> tag's type attributes has to be specified
  • Types of button includes:
    • button: A regular button looking button with no default behavior. Can be associated with element's event using javascript or other client-side script
    • reset: Used with forms, reset all controls to default value
    • submit: Used with forms, submit the form data to server
  • Buttons can be disabled with attributesdisabled
