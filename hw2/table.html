The <table> tag

The <table> element represents an organized collection of data.

Example 1

This table contains absolutely no styling. There is no special spacing of the data, and there are no borders.

Name Age
John Smith Jane Doe
21 23

All attributes unique to the table tag have been deprecated in favor of CSS stylesheets.

Example 2

This table has been styled with CSS to have borders and a background color.

Name Age
John Smith Jane Doe
21 23
