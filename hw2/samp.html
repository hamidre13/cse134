The <samp> element

The content inside of a <samp> element is meant to indicate sample output from a computer program.

Example 1

This is text in a <p> tag. If you wrote a function 

 function inc(num) {
    return num++;
}

and passed the value 6, you would expect to get the value

7.

The function is written within a <pre> element, and the output is written in a <samp> element.
