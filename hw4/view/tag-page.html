<input>

Definition

The <input> tag defines an input field where the user can enter data. This element is used inside the <form> tag to provide interactive controls for user data entry.

Compatible Attributes
  • type
  • accept
  • autocapitalize

Specifies the type of input the user can enter, and provides the correct controls. This attribute can be numerous values.

If input type is file, then this indicates what type of files are accepted by the server.

Nonstandard, used by iOS and Safari. Controls whether and how the data entered should be capitalized as it is entered by the user.
